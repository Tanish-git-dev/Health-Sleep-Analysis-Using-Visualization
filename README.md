# Health and Sleep Analysis

## Overview
This project analyzes the relationship between lifestyle factors and sleep quality using Python and data visualization techniques. The analysis explores how stress levels, physical activity, age, BMI, and sleep disorders impact sleep patterns.
## Dataset
The dataset contains information related to:
* Age
* Sleep Duration
* Quality of Sleep
* Stress Level
* Physical Activity Level
* BMI Category
* Sleep Disorders
## Analysis Performed
* Correlation between stress level and sleep quality
* Correlation between physical activity and sleep quality
* Correlation between age and sleep quality
* Analysis of REM sleep cycles
* Impact of BMI on sleep disorders
* Comparison of sleep duration for individuals with and without sleep disorders
## Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
## Key Findings
The analysis reveals significant relationships between lifestyle factors and sleep quality, highlighting the effects of stress, physical activity, BMI, and sleep disorders on overall sleep health.
